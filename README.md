## System requirements
 * YooTheme Zoo Component `^3.3.28`
 * Joomla! CMS `^3.8.3`
 * Joomla! CMS `4.0` (maybe...)
 * PHP `>= 7.0` is recommended with modules (mbstring, xml, json, opcache, mysqli)
